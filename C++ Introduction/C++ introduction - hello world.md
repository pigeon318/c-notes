
# Hello world

## example  of a hello world c++ program

```cpp 
         #include <iostream> // includes the input/output library 
         int main() { // main function - program starts here 
         std::cout << "Hello World"; // prints text 
         return 0; // tells OS program finished ok 
         }
```

## output

you can output a string in c++ with the *std::cout* stream  followed by << which i an operator that comes after and then "Your string\n" "Your string" is the text you want outputted and \n is a special character that indicates a new line
**ALL C++ LINES EXCEPT FOR IMPORTS AND FUNCTION CREATIONS MUST END IN ;** 
### patterns
in c++ you can make patterns by using the \n for example the code
```cpp 
int main()
{
  std::cout << "      1\n";
  std::cout << "    2 3\n";
  std::cout << "  4 5 6\n";
  std::cout << "7 8 9 10\n";
}   
```


would output the pattern      1 
                       2 3
                    4 5 6 ... etc.
