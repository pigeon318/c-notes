## Notes (in learning order)
 
### Basics
 
1. **Hello World** — Program structure, `#include`, `main()`, `std::cout`, output, newline characters, patterns.
2. **Variables** — Data types (`int`, `double`, `char`, `string`, `bool`), declaring and initialising variables, arithmetic operators, outputting variables, chaining output, user input with `std::cin`.
3. **Conditionals** — `if`, `else`, `else if`, relational operators (`==`, `!=`, `>`, `<`, `>=`, `<=`), switch statements.
4. **Logical Operators** — `&&` (and), `||` (or), `!` (not), truth tables.
5. **Loops and Errors** — `while` loops, `for` loops, compile-time errors, link-time errors, run-time errors, logic errors.
6. **Vectors** — Creating vectors, initialising, indexing, `.push_back()`, `.pop_back()`, `.size()`, iterating over vectors with loops.
7. **Functions** — Declaring and defining functions, `void`, return types, parameters and arguments, multiple arguments.
8. **Functions Pt2** — Scope, multi-file programs, header files, inline functions, default arguments, function overloading, function templates.
9. **Classes and Objects** — Classes as blueprints, attributes and methods, public and private access, defining methods outside the class, creating objects, header and source file structure.
